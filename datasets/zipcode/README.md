## データ出典

[日本郵便](https://www.post.japanpost.jp/zipcode/dl/utf-zip.html)が公開している全国の郵便番号データです。

## テーブル: mart_zipcode

郵便番号と住所の対応データです。

- lg_code: 全国地方公共団体コード（6桁）
- zipcode: 郵便番号（7桁）
- old_zipcode: 旧郵便番号（5桁）
- prefecture / city / town: 都道府県 / 市区町村 / 町域（漢字）
- prefecture_kana / city_kana / town_kana: 都道府県 / 市区町村 / 町域（カタカナ）
