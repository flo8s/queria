# e-Stat 小地域データ A-K カテゴリカバレッジ対応表

調査日: 2026-03-02

## 概要

e-Stat API の getStatsList (searchKind=2) から取得した小地域（町丁・字等）統計データの、
市区町村版 A-K カテゴリとの対応状況。

全 7,535 テーブル中:
- 小地域: 2,632 テーブル（4調査）
- 地域メッシュ: 4,903 テーブル

小地域データの提供元は 国勢調査 と 経済センサス系 の 4 調査のみ。
市区町村版の「社会・人口統計体系」とは異なるデータソース。

## 小地域データ提供調査の一覧

### 国勢調査 (00200521) - 2,209テーブル

最新: 2020年（令和2年） / 過去: 2000, 2005, 2010, 2015

ユニークなテーブルパターン（最新年度）:
1. 男女別人口総数及び世帯総数
2. 年齢（5歳階級、4区分）別、男女別人口
3. 世帯人員別一般世帯数
4. 世帯の家族類型別一般世帯数
5. 世帯の経済構成別一般世帯数
6. 住宅の所有の関係別一般世帯数
7. 住宅の建て方別世帯数
8. 産業（大分類）別及び従業上の地位別就業者数
9. 職業（大分類）別就業者数

過去年度のみ:
- 在学学校・未就学の種類別在学者数・未就学者数（2000年のみ）
- 65歳以上親族のみの一般世帯数（2000年のみ）

### 経済センサス-基礎調査 (00200552) - 188テーブル

最新: 2014年（平成26年） / 過去: 2009

ユニークなテーブルパターン:
1. 産業（大分類）別・従業者規模別全事業所数及び男女別従業者数
2. 経営組織別民営事業所数及び従業者数

### 経済センサス-活動調査 (00200553) - 141テーブル

最新: 2012年（平成24年）

ユニークなテーブルパターン:
1. 産業（大分類）別民営事業所数及び男女別従業者数
2. 経営組織（4区分）別民営事業所数及び従業者数
3. 従業者規模（6区分）別民営事業所数及び従業者数

### 事業所・企業統計調査 (00200551) - 94テーブル ※廃止調査

最新: 2001年（平成13年）

2009年に経済センサス-基礎調査に統合され廃止。

## 構造上の注意点

市区町村版との主な違い:

1. テーブルが都道府県別に分割されている
   - 1つのテーブルパターンにつき 47 の statsDataId がある
   - 全国データを得るには 47回の API 呼び出しが必要

2. time 軸がない
   - 市区町村版は time 軸で複数年データを格納しているが、
     小地域版は調査年ごとに別テーブルとなっている

3. cat02 軸（秘匿フラグ）がある
   - 国勢調査のテーブルには cat02 として秘匿地域・合算地域フラグが付与される
   - 値: 無し / 合算 / 秘匿

4. area コードの粒度
   - 市区町村コード（5桁） → 大字（9桁） → 町丁・字等（11桁）の階層構造
   - 1都道府県あたり数千〜2万5千の area クラスが含まれる
